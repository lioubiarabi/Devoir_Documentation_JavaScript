# Types de Données

## Types primitifs vs types référence

there are 7 primitifs:
`string` `number` `bigint` `boolean` `undefined` `null` `symbol`

and the data store directly to them;

and there are types référence, which are objects, and data store with a refrence or pointer to a memory address

for example:
`object` `functions` `arrays` ...


## typeof

we can use typeof so it can return the data type as a string

for example:
```
typeof "Bonjour"        // "string"
typeof 42               // "number"
typeof 123n             // "bigint"
typeof true             // "boolean"
```
